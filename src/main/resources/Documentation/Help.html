DrivenByMoss - Bitwig Studio Extension for several controllers

DrivenByMoss is a Bitwig Studio extension, which supports many controllers and protocols. Furthermore, it brings some tools and utilities. DrivenByMoss is written by Jürgen Moßgraber also known as MOSS.
